## 电脑配置

|       规格        |                   详细信息                    |
|:---------------:|:-----------------------------------------:|
|      电脑型号       |               acemagic-ck11               |
|      操作系统       |              macOS `Sonoma`               |
|       处理器       |          AMD 锐龙 R7-5800U 8核16线程           |
|       内存        |         4 GB DDR4 2400MHz （免费拿的）          |

|       硬盘3       |             可接SATA 2.5寸硬盘/SSD             |
|       核显        |  Radeon Vega Graphics<br /> |

|       声卡        |             USB Audio Device              |
|      无线网卡       | m.2 NGFF插槽，默认出厂为 `RTL 8852BE` 已更换为`AX210` |
